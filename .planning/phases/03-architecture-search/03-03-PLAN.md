<task type="auto">
  <name>Task 3: Extract D1 best config and merge into search_results.json on main</name>
  <files>search_results.json</files>
  <action>
    Extract best D1 config from results_d1.tsv and merge into search_results.json on main.

    Steps:
    1. Confirm branch: git branch --show-current (should be autoresearch/arabic-d1)

    2. Write and run extraction script:
       Write extract_d1.py (temporary):

       import csv, json, subprocess
       from pathlib import Path

       def extract_best(tsv_path):
           best_row = None
           with open(tsv_path, newline="") as f:
               reader = csv.DictReader(f, delimiter="\t")
               for row in reader:
                   if row["status"] != "keep":
                       continue
                   if best_row is None or float(row["val_bpb"]) < float(best_row["val_bpb"]):
                       best_row = row
           if best_row is None:
               raise ValueError("No 'keep' rows found")
           commit = best_row["commit"]
           train_src = subprocess.check_output(["git", "show", f"{commit}:train.py"], text=True)
           config = {"commit": commit, "best_val_bpb": float(best_row["val_bpb"])}
           for var in ["DEPTH", "WINDOW_PATTERN", "HEAD_DIM", "ASPECT_RATIO", "TOTAL_BATCH_SIZE"]:
               for line in train_src.splitlines():
                   if line.strip().startswith(f"{var} ="):
                       config[var.lower()] = line.split("=", 1)[1].strip().strip('"')
                       break
           return config

       d1_result = extract_best("results_d1.tsv")
       print(json.dumps(d1_result, indent=2))
       # Merge into search_results.json on disk (will switch to main after)
       import tempfile, os
       with tempfile.NamedTemporaryFile("w", delete=False, suffix=".json") as f:
           json.dump(d1_result, f, indent=2)
           tmp = f.name
       print(f"d1 result written to {tmp}")

       Run: uv run python extract_d1.py
       Note the temp file path.

    3. Switch to main and merge:
       git checkout main
       python -c "
       import json
       from pathlib import Path
       d1_result = json.loads(open('{tmp_path}').read())
       p = Path('search_results.json')
       existing = json.loads(p.read_text())
       existing['d1'] = d1_result
       p.write_text(json.dumps(existing, indent=2))
       print('merged d1 into search_results.json')
       print(existing)
       "
       rm extract_d1.py

    4. Commit on main:
       git add search_results.json
       git commit -m "feat(03): add D1 search results to search_results.json"
