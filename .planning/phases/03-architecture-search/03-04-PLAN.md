  <files>search_results.json</files>
  <action>
    Extract D2 best config, complete search_results.json with all three conditions, run the full test suite to verify Phase 3 is done.

    Steps:
    1. Confirm branch: git branch --show-current (should be autoresearch/arabic-d2)

    2. Extract D2 best config:
       Write extract_d2.py (temporary):

       import csv, json, subprocess
       from pathlib import Path

       def extract_best(tsv_path):
           best_row = None
           with open(tsv_path, newline="") as f:
               reader = csv.DictReader(f, delimiter="\t")
               for row in reader:
                   if row["status"] != "keep":
                       continue
                   if best_row is None or float(row["val_bpb"]) < float(best_row["val_bpb"]):
                       best_row = row
           if best_row is None:
               raise ValueError("No 'keep' rows found")
           commit = best_row["commit"]
           train_src = subprocess.check_output(["git", "show", f"{commit}:train.py"], text=True)
           config = {"commit": commit, "best_val_bpb": float(best_row["val_bpb"])}
           for var in ["DEPTH", "WINDOW_PATTERN", "HEAD_DIM", "ASPECT_RATIO", "TOTAL_BATCH_SIZE"]:
               for line in train_src.splitlines():
                   if line.strip().startswith(f"{var} ="):
                       config[var.lower()] = line.split("=", 1)[1].strip().strip('"')
                       break
           return config

       d2_result = extract_best("results_d2.tsv")
       print(json.dumps(d2_result, indent=2))
       import tempfile
       with tempfile.NamedTemporaryFile("w", delete=False, suffix=".json") as f:
           json.dump(d2_result, f, indent=2)
           print("d2 temp:", f.name)

       Run: uv run python extract_d2.py
       Note the tmp path.

    3. Switch to main and merge to complete search_results.json:

       existing = json.loads(p.read_text())
       existing['d2'] = d2_result
       p.write_text(json.dumps(existing, indent=2))
       print('Complete search_results.json:')
       print(json.dumps(existing, indent=2))
